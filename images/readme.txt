Directory structure for images:

/images
  /hero - For hero background images
  /menu - For menu item images
  /about - For about section images

Image dimensions:
- Hero images: 1800x800px
- Menu item images: 600x400px (will display as 300x200)
- About section images: 600x400px

All images should be optimized for web (jpg or webp format, compressed)
